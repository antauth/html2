# Using the Visual Studio (VS) Code editor

1. Double-click on the Visual Studio Code shortcut.
3. In the VS Code menu, select File > Open Folder...
4. Select the desired folder (the one with your html in it) on the Desktop.
5. Double-click on the `index.html` (or other html) file in the left sidebar.
6. In the editor, right-click anywhere in the `index.html` (or other html) file and select **Show Preview**.

# Install VS Code

If you want to use VS Code on your local computer, you can do so!

[Click here to visit the download page.](https://code.visualstudio.com/download)

## Use Live Preview
If you want to use live preview (as mentioned in Step 6 above), install the Live Preview by Microsoft. [You can find instructions on how to install extensions here.](https://github.com/microsoft/vscode-docs/blob/main/docs/getstarted/extensions.md)
